Student Performance Dashboard (Power BI)
Overview

This Power BI dashboard presents a simple analysis of student academic performance using attendance, internal marks, assignment marks, and final grades. It helps in understanding overall performance and trends at a glance.
Key Highlights

	•	Total number of students

	•	Average attendance percentage

	•	Average internal marks

	•	Average assignment marks
Visualizations Included

	•	Department-wise Average Internal Marks – compares performance across departments

	•	Semester-wise Average Internal Marks – shows performance trend across semesters

	•	Final Grade Distribution – displays the count of students in each grade category
Tools Used

	•	Microsoft Power BI

	•	Student performance dataset
Purpose

This dashboard is created for academic analysis and learning purposes to demonstrate data visualization and insights using Power BI.
